# Telegram Smoke Test

1. Send `/start` and check the main menu.
2. Create a client.
3. Open the client list and open the new client.
4. Add a car to the client.
5. Create a booking for that client and car.
6. Open "Today" or "Tomorrow" and check the booking appears.
7. Open the booking card.
8. Start a repair from the booking.
9. Change labor price.
10. Add a part to stock.
11. Add the part to the repair.
12. Record a payment.
13. Complete the repair.
14. Check low stock.
15. Start any multi-step action, press `/cancel`, and verify the dialog resets to the main menu.
